# Advent of Code Solutions in Java

This repository contains solutions for the Advent of Code (AoC) event, written in Java. 

## Prerequisites

- JDK 17

## Project Structure

Each day's puzzle solution is located in its respective package labeled by the day (for example, `day1`, `day2`, etc.). 

## Running the solutions
